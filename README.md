# 📁 README

# StopEat - BMI 기반 식단 관리 웹 서비스

## 📌 프로젝트 소개

웹프로그래밍 수업에서 진행한 개인 프로젝트입니다.

HTML, CSS, JavaScript, PHP, MySQL을 활용하여 회원가입 및 로그인 기능, BMI 계산, 체중 및 식단 기록 기능을 제공하는 웹 서비스를 개발하였습니다.

웹 개발을 처음 배우며 진행한 프로젝트로, 웹 서비스의 기본 구조와 데이터베이스 연동 과정을 직접 구현하며 학습하였습니다.
